This is my take on a basic ugly retro calculator. The logic and DOM magic is all coded in vanilla JavaScript and it's styled using basic CSS. I wanted it to operate as closely as possible to my iPhone calculator because that's what I had on hand. You can read more on github.


While this technically works on mobile, it's more fun on desktop with a keyboard that has a tenkey. Esc key will clear the calculator. The hyphen/underscore key will toggle the number sign to positive or negative. Enjoy!
